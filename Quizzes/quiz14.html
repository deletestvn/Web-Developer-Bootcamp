Arrays Problem Set

Use console to try the following functions:

  • printReverse(): takes an array as an argument and prints out the elements in the array in reverse order
  • isUniform(): takes an array as an argument and returns true if all elementsin the array are identical
  • sumArray(): takes an array of numbers and returns the sum of all numbers in the array
  • max(): takes an array of numbers and returns the maximum number in the array
